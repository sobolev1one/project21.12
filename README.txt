Индивидуальный проект
Программа, сохраняющая изображения из различных источников и добавляющая на них водяной знак. Поддерживает яндекс диск.
Файл links содержит ссылки на изображения из моего яндекс диска, на которых можно проверить работоспособность программы
Программа на вход принимает ссылку на изображение
Результатом работы программы становятся два изображения. Одно из них оригинальное, второе содержит водяной знак
